# Content Analyzer P2 体验优化设计

日期：2026-03-10

## 目标
1. 返回列表后恢复原滚动位置。
2. 列表状态 URL 持久化（tab/orderBy/page）。
3. 预览面板支持键盘与手势切换。

## 方案
- 在进入详情前记录列表上下文（tab/orderBy/page/scrollY）。
- 列表组件初始化时读取上下文并恢复滚动。
- 列表状态变化时同步写入 URL query（replace，不触发整页刷新）。
- 预览面板增加：Esc 关闭、左右方向键切换、触摸左右滑切换媒体。

## 验证
- 详情返回后不跳顶部，滚动位置恢复。
- 刷新后 tab/orderBy/page 参数保留。
- 预览面板键盘和手势切换可用。
